Viewer

NameSpace: Blazor3D.Viewers

Blazor3D viewer component.

Methods

SetCameraPositionAsync(Vector3, Vector3)

Sets the camera position to specified Vector3 value.

Returns: Task

Parameters
position
New Vector3 position.
lookAt
New Vector3 camera target point coordinates.
RemoveByUuidAsync(Guid)
Removes object from scene by it's unique identifier.

Returns: Task

Parameters
uuid
Object's unique identifier.
ClearSceneAsync()
Clears scene.

Returns: Task
Import3DModelAsync(Import3DFormats, string, string, Guid})

Imports 3D model to scene.

Returns: Guid of the loaded item

Parameters
format
Import3DFormats format of 3D model.
objUrl
URL of the 3D model file
textureUrl
URL of the texture file
uuid
UUID of the object to be loaded. Nullable. If not specified, the new Guid is genrated.
GetObjectByUuid(Guid, Object3D})

Recursively finds object by it's uuid in collection.

Returns: Found object or null

Parameters
uuid
Object's uuid
children
Children collection. Usually it's Scene.Children

Properties

ViewerSettings

ViewerSettings parameter of the component.
Scene

Scene parameter of the component. Default is empty scene.
UseDefaultScene

If true and there is no children objects in the scene, then adds the default lights and box mesh. Default value is false.
Camera

PerspectiveCamera used to display the scene.
OrbitControls

OrbitControls used to rotate, pan and scale the view.

Events

ObjectSelected
Raises when user selects object by mouse clicking inside viewer area.
ObjectLoaded
Raises after complete loading of imported file content.
